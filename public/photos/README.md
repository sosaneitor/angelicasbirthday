# Fotos para el Carrusel

Coloca aquí las fotos que quieras mostrar en el carrusel.

## Nombres esperados:
- `1.jpg` — Primer viaje juntos
- `2.jpg` — Momentos especiales
- `3.jpg` — La llegada de Cyan
- `4.jpg` — Foto familiar
- `5.jpg` — Momentos de amor
- `6.jpg` — Juntos somos más

## Formato:
- Recomendado: JPG o PNG
- Tamaño ideal: 800x1000px (portrait) o 1000x800px (landscape)
- Aspect ratio: libre, se adaptan automáticamente

## Para activarlas:
Una vez coloques las fotos, edita `src/components/steps/PhotoCarousel.tsx` 
y cambia los placeholders por tags `<img>` reales.
